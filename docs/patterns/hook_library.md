# Hook Library — 훅 라이브러리

첫 0.5~1초 안에 스크롤을 멈추게 하는 훅(Hook)의 원칙과 후보 모음입니다.
관련 결정: [DL-005](../operations/decision_log.md)

---

## 훅 원칙

> **모든 영상은 첫 0.5~1초 안에 스크롤을 멈추게 해야 한다.**

- 훅은 "예쁨"이 아니라 "멈춤"을 만든다.
- 사람이 쉽게 찍을 수 없는 장면일수록 강하다.
- HARIN은 고정, 세계가 극적으로 변할 때 훅이 발생한다.

## HARIN용 훅 후보

| 훅 | 개념 |
|----|------|
| **Gravity Break** | 중력이 무너진다 — 하린만 그대로, 세계가 떠오른다 |
| **World Collapse** | 세계가 붕괴한다 — 하린은 중심축으로 남는다 |
| **Nature Obeys** | 자연이 하린에게 반응/복종한다 |
| **Silent Before Impact** | 충돌 직전의 정적 — 소리가 사라지고 한 박자 멈춘다 |

## 금지 (하지 말 것)

- ❌ 단순 예쁜 모델 워킹
- ❌ 카페 / 창가 / 엘리베이터 감성
- ❌ 사람이 쉽게 찍을 수 있는 평범한 장면

## 승격 규칙

- 실제 제작에서 성공이 검증된 훅만 `proven_recipes` 로 승격한다.
- 실패한 훅은 [retrospectives](../retrospectives/) 에 원인과 함께 기록한다.
