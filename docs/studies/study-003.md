## 핵심 학습 (Extracted Concepts)

### 1. Brand First, Campaign Second
- 캠페인보다 **브랜드가 먼저**다. 브랜드 정체성·톤을 먼저 확정하고, 개별 캠페인은 그 위에서 파생된다.

### 2. Asset Library Before Content Production
- 콘텐츠 제작에 앞서 **에셋 라이브러리를 먼저 구축**한다. 재사용 가능한 에셋이 준비된 후에 생산에 들어간다.

### 3. Checklist-Based AI Production
- AI 제작은 즉흥이 아니라 **체크리스트 기반**으로 진행한다. 단계별 확인 항목을 따라 일관성을 유지한다.

### 4. Iterative Prompt Refinement
- 프롬프트는 한 번에 완성하지 않는다. **반복 정제(iterative refinement)** 로 점진적으로 개선한다.

### 5. One Asset System → Many Ad Outputs
- 하나의 **에셋 시스템**에서 **다수의 광고 산출물**을 파생한다. 시스템 1개 → 아웃풋 N개.
