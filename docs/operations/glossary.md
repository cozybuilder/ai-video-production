# Glossary

> Register가 아니라 **참조 문서**다. 용어 정의만 관리한다.

---

- **Rule** — 실패에서 검증된 규칙. 생애: H → T → C → P → R. ([operating_rules.md](operating_rules.md))
- **Pattern** — 성공 사례 축적. 더 좋은 방법이 나오면 교체 가능. ([pattern_register.md](../patterns/pattern_register.md))
- **Gate** — 실수를 막는 자동 연결 장치. ([gates.md](gates.md))
- **Register** — 현재 상태(Current State)를 담는 문서.
- **Decision Log** — 결정의 역사(History)를 담는 문서. ([decision_log.md](decision_log.md))
- **Identity Asset** — Soul / Brand / Palette / Logo / Voice / Typography.
- **Output Asset** — Scene / Keyframe / Motion / Background / Object.
- **Locked Asset** — 동결되어 재사용되는 자산.
- **Hypothesis** — Study가 생성하는 테스트 가능한 가설.
- **Register Action** — 회고에서 기록하는 Register 변경(Rule/Pattern/Asset/변경없음).
