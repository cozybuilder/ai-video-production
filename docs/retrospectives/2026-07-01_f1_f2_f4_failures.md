# Retrospective — F1 / F2 / F4 실패 회고 (2026-07-01)

관련 결정: [DL-008](../operations/decision_log.md)

---

## 성격 규정

- F1 / F2 / F4 영상은 **공식 제작물이 아니다.**
- **실패 실험 / 교훈**으로만 기록한다.
- 제작은 SSOT 문서 저장 이후 **처음부터 다시 시작**한다.

## 실패 원인

- 이미지 승인 전에 영상을 생성함
- 한 클립에 이벤트를 과다하게 담음
- 얼굴 / 의상 / 세계관 불일치
- 쓰나미 방향 오류
- 도시 오브젝트가 해안까지 남아있는 오류
- 문서화 없이 채팅으로만 진행함
